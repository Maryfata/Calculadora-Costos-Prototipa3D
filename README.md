# Calculadora de Costos - Prototipa3D

Herramienta profesional de ingeniería para el cálculo de costos operativos en impresión 3D. 

## Características
- **Cálculo de insumos:** Luz, material, desgaste de máquina y mantenimiento.
- **Costos de tiempo:** Cálculo automatizado según horas de diseño y complejidad de pieza.
- **Segmentación de precios:** Precios diferenciados para Estudiantes y Público General.
- **Módulos opcionales:** Incluye empaque, traslado, impuestos (IVA) y comisiones bancarias.

## Cómo usarla
1. Asegúrate de tener instalado Python.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta el archivo en Jupyter Notebook o VS Code.
